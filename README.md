### What works and What doesn't or WIP:

- [x] Intel HD 620 iGPU HDMI Output
- [x] ALC282 Internal Speakers
- [x] ALC282 Native Combojack

- [x] All USB Ports 
- [x] SpeedStep / Sleep / Wake
- [x] Touchpad with gesture
- [x] Brightness Key
- [x] Wi-Fi
- [x] Realtek RTL8111 LAN
- [x] Cardreader
- [x] ACPI Battery
